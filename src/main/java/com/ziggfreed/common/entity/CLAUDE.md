# entity/ - player/entity model swap primitives

Router for `com.ziggfreed.common.entity`. Mod-agnostic helpers for mutating a rendered entity.

- **[`PlayerModelService`](PlayerModelService.java)** - swap a PLAYER's rendered model and restore it, mirroring the engine `builtin.model.pages.ChangeModelPage` (`ChangeModel` / `ResetModel`). `apply(ref, store, modelId, scale)` resolves the `ModelAsset` (missing/invalid = no-op, warned once per id), builds a scaled `Model` via `Model.createScaledModel` (scale floored to a positive value so the engine's `scale > 0` guard never throws), and replaces the immutable `ModelComponent`; returns `true` only when applied. `restore(ref, store)` rebuilds the player's CORRECT cosmetic model from `PlayerSkinComponent` via `CosmeticsModule.get().createModel(skin.getPlayerSkin())` then `setNetworkOutdated()` - NOT a cached `Model` (a cached one holds a stale reference); idempotent, safe on any player (no-op without a skin component). `modelExists(modelId)` for early preset validation. **WORLD-THREAD ONLY** (the caller guarantees the thread for every `Store`/`Ref` read + `putComponent`; this primitive does not hop itself); every engine call is try-guarded so a bad ref / missing asset degrades to a no-op and never throws into the caller.
